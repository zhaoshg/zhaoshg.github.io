linux 查找目录或文件

发表于 2017-04-21 | 分类于 linux | | 阅读次数
字数统计 2,103 | 阅读时长 9

查找目录：find /（查找范围） -name '查找关键字' -type d
查找文件：find /（查找范围） -name 查找关键字 -print
如果需要更进一步的了解，可以参看Linux的命令详解。 这里摘抄如下：

# linux # 转载 # find # 查找
